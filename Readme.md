# TP-AAR: Linear-Time Multi-View Attributed Graph Clustering via Tensor Pre-Fusion and Adaptive Anchor Refinement

This is the implementation for our paper: **TP-AAR: Linear-Time Multi-View Attributed Graph Clustering via Tensor Pre-Fusion and Adaptive Anchor Refinement**.

## Quick Demo to Reproduce Clustering Results

To facilitate quick verification of our results, we provide a `demo.py` script and the pre-computed self-representation matrix (`ACM_Z.pkl`) for the **ACM** dataset. You can run this script to directly obtain the final clustering metrics reported in our paper without retraining the entire model.
